as3proto-osx

protobuf for Actionscript 3 on OSX platform

该项目搭建了AS3 protobuf所需的Windows运行环境以及AS3类库，整合了Google protobuf和protobuf-actionscript3第三方插件资源

使用方法

  1. 编译安装protoc命令行

    在Terminal里面进入sdk目录，运行下面脚本

    ./configure
make
make check
sudo make install
  2. 使用protoc生成代码
    protoc --proto_path=proto --as3_out=output proto/hello.proto
