Packagecom.google.protobuf
Classpublic final class WireFormat
InheritanceWireFormat Object

WireFormat provides a static object from which to retrieve important encoding information regarding a particular tag. Used primarily in the CodedInputStream and the CodedOutputStream; it really should not be needed oustide of these clases.
